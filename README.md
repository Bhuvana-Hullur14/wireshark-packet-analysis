# wireshark-packet-analysis
Network Packet Analysis using Wireshark (HTTP, DNS, TCP,)


## Objective
To capture and analyze network packets using Wireshark.

## Tools Used
- Wireshark
- Kali Linux / Windows

## Protocols Analyzed
- HTTP
- DNS
- TCP

## Key Observations

### HTTP
- Observed GET request
- Response Status: 200 OK

### DNS
- DNS Query for google.com
- Resolved IP address

### TCP
- Observed 3-Way Handshake (SYN, SYN-ACK, ACK)


## Skills Demonstrated
- Network Traffic Analysis
- Protocol Inspection
- Packet Filtering
- Understanding TCP/IP Stack
